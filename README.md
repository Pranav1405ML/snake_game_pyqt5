# 🐍 Snake Game (PyQt5)

A GUI-based Snake Game built using Python and PyQt5.

## 🚀 Features
- Real-time score tracking
- Smooth movement
- Background music
- Restart option
- Clean GUI interface

## 🛠 Tech Stack
